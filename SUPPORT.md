# Support

## Getting Help

**GitHub Issues**  
For bugs or unexpected behavior, open an issue. Use the bug report template and include enough context to reproduce the problem.

**Email**  
For general questions or anything that isn't a bug — `support.sxnnyside@sxnnysideproject.com`

**Computtite Website**  
[computtite.sxnnysideproject.com](https://www.computtite.sxnnysideproject.com)

**Sxnnyside Project**  
[sxnnysideproject.com](https://www.sxnnysideproject.com)

---

## Before Asking

- Check the README — installation, update process, and features are documented there.
- Search existing issues before opening a new one.
- Include your platform (macOS / Windows / Linux), Computtite version, and a clear description of the problem.

---

## What This Isn't

This is not a priority support channel. Response times are not guaranteed.  
For security vulnerabilities, see [SECURITY.md](SECURITY.md) — do not open a public issue.

---

## Supporting Computtite

Computtite is free and voluntarily funded. If it helps your work, consider supporting development on [Patreon](https://www.patreon.com/c/sxnnysideproject/membership).
